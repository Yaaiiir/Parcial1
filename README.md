# Examen de la primera parcial
¡Bienvenidos! Este es un espacio de trabajo colaborativo.

## 🚀 Cómo contribuir
Para mantener el orden, sigue estos pasos:

1. **Clona el repositorio:** `git clone <url-del-repo>`
2. **Crea tu rama:** Usa un nombre descriptivo, por ejemplo: `git checkout -b feature/nombre-del-estudiante`
3. **Trabaja libremente:** Haz tus commits en tu rama.
4. **Sube tu rama:** `git push origin nombre-de-tu-rama`
5. **Pull Request:** Si quieres integrar algo a la rama principal (main), abre un PR para que lo revisemos.
6. **Para bajar los cambios:** `git fetch --all`
7. **Para moverse a su propia rama:** `git checkout mi-rama-personal`
8. **Para asegurar que su rama esté al dia con lo que hay en el servidor:** `git pull origin mi-rama-personal`
